即插即用,不需要对源码进行任何修改!

## 环境
pip install grad-cam==1.4.8 -i https://pypi.tuna.tsinghua.edu.cn/simple

## 注意事项
1. yolov5是在v7.0进行编写和测试的。
2. yolov7是在2023.10.1号的版本进行编写和测试的。
3. yolov8是在2024.1.31号的版本进行编写和测试的。
4. 建议在新版本下进行使用，旧版本可能会有报错，需要自行解决。
